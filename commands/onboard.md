# /onboard

Interactive walkthrough of the project vault and codebase.

## Protocol
1. Dispatch `agents/onboarding.md`
2. Onboarding Guide walks through: project structure, architecture, key files, build process, personas
3. Adapts to audience: developer, AI agent, or operator

Usage:
- `/onboard` — full walkthrough
- `/onboard [topic]` — focused walkthrough (e.g., `/onboard auth`, `/onboard payments`)
- `/onboard agent` — optimized for a new AI agent session
