### OS-ADL-004: ~~LightRAG for Knowledge Graph~~
**Status:** SUPERSEDED by OS-ADL-023 | **Date:** 2026-03-30 | **Superseded:** 2026-04-04 | **Domain:** architecture
**Original decision:** LightRAG (`lightrag-hku`) with MCP bridge provides the OS's knowledge graph.
**Why superseded:** (1) LightRAG benchmarks inflated by evaluation bias — independent study found NaiveRAG comparable after correcting for position and length bias. (2) Microsoft's LazyGraphRAG outperformed LightRAG in all 96 comparison tests. (3) Python dependency in a Rust/Tauri desktop app = external process, Python install requirement, extra failure mode. (4) CVE-2026-30762 (hardcoded JWT secret) disclosed 2026-04-04 — signals immature security posture. (5) No public production deployment at scale documented. (6) Vault content is structured (YAML frontmatter, tags, typed traces) — SQL exploits this structure better than graph RAG. (7) SQLite already in the project at V15 — adding sqlite-vec is one C extension, not a new database. See OS-ADL-023 for replacement.
