# Retrospective Template

> **Standard format for retro session output.** Produced during Phase 4 of the Retrospective Protocol.

---

## Template

```markdown
# Retrospective: [Scope] — [Date]

## Scope
[What was reviewed — batch range, layer, or incident]

## Participants
[Which personas participated]

## Metrics
| Metric | Value |
|--------|-------|
| Batches completed | ___ |
| Steps completed vs planned | ___ / ___ |
| Findings discovered (by severity) | CRIT: ___ HIGH: ___ MED: ___ LOW: ___ |
| Previous retro commitments completed | ___ / ___ |
| Verification pass rate | ___% |

## What Went Well
[Key themes with specific examples]

## What Didn't
[Key themes with specific examples]

## Previous Retro Follow-Through
| Commitment | Owner | Status |
|-----------|-------|--------|
| [description] | [persona] | Done / In Progress / Not Addressed |

## Key Insights
[2-3 most important learnings from this period]

## Commitments
1. [Action] — Owner: [persona] — By: [date] — Success: [criteria]
2. [Action] — Owner: [persona] — By: [date] — Success: [criteria]
3. [Action] — Owner: [persona] — By: [date] — Success: [criteria]

## Preparation Notes for Next Phase
[What needs to happen before the next batch/layer starts]
```
